# my-fullstack-jurney

A structured, self-paced guide for anyone who wants to become a Full Stack Developer — notes, assignments, and milestones organized by topic

> **Status: Work in Progress** — this repo is actively being updated as I learn. Expect new folders, revised notes, and reorganizations over time.

---

## 🗺️ How This Repo is Organized

- **`01-Frontend/`** — Everything needed to go from "what is the internet" to writing clean HTML, CSS, and JavaScript, plus a milestone project building static pages.
- **`02-Backend/`** — Deeper dives into JavaScript, version control, and backend-relevant networking concepts.
- **`00-Materials/`** — Shared resources, references, and external materials used across topics.

Each numbered folder is meant to be followed roughly in order, with `Notes` and `Assignments` subfolders separating theory from practice. Milestone folders (like `04-Milestone-Static-Pages`) mark checkpoints where concepts are combined into a small project.

---

>To provide a clear, practical path for aspiring full stack developers
